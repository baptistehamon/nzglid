# New Zealand Gridded Land Information Dataset (NZGLID)

[![DOI](https://zenodo.org/badge/DOI/10.5281/zenodo.16809254.svg)](https://doi.org/10.5281/zenodo.16809254)

This repository contains the code used to create the New Zealand Gridded Land Information Dataset (NZGLID). The NZGLID integrates data from various sources to offer detailed spatial information about the land, soil, terrain, and environment of New Zealand's mainland. The dataset is available at spatial resolutions of 5 km and 1 km (WGS84). The input data were processed to achieve the final resolutions by rasterizing geospatial vector layers and reprojecting and performing bilinear resampling of raster layers. The dataset is available on [Zenodo](https://doi.org/10.5281/zenodo.16809254).
